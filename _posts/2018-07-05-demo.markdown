# 1、安装下载

```
官网 https://www.sonatype.com/
```
下载最新版本的nexus,上传到Linux后解压运行

```
在bin目录，
    启动：./nexus start
    关闭：./nexus stop
    重启：./nexus restart
    状态：./nexus status
```
